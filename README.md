# nrempel-plugins

My Claude Code plugin marketplace.

## Installation

Add this marketplace:

```
/plugin marketplace add nrempel/claude-plugins
```

Browse available plugins:

```
/plugin
```

Install a plugin:

```
/plugin install plugin-factory@nrempel-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [plugin-factory](https://github.com/nrempel/claude-plugin-factory) | Create Claude Code plugins, skills, agents, and hooks |
